# Project folder icon suggestions

**Priority:** Medium

## Conversation

### Assistant

A common icon to signify a **"Project Folder"** usually combines two visual ideas:  

1. **Folder shape** – the universal symbol for storage or grouping of files.  
2. **Project indicator** – an overlay or embedded element that hints at work-in-progress, content type, or context.  

Here are some standard variations:  

- **📁 Folder + Document/Page** – represents a folder containing project files.  
- **📂 Folder + Gear ⚙️** – for project folders tied to settings, builds, or configurations.  
- **📁 Folder + Checkmark ✔️** – for completed or approved projects.  
- **📁 Folder + Kanban/Board/Grid** – for project management context.  
- **📁 Folder + Star ⭐** – for “important” or “pinned” project folders.  
- **📁 Folder + Hammer/Wrench 🛠️** – for development or construction projects.  

In icon libraries like **SF Symbols**, **Material Icons**, or **Feather Icons**, you’ll often see things like:  
- `folder.fill` + overlay (`gear`, `doc`, `chart.bar`, etc.)  
- Material Icons: `folder_special`, `folder_open`, `folder_shared`, `workspaces`  
- Feather: `folder`, `folder-plus`, `folder-minus`  

If you want a **universal, minimal look**, the safest bet is:  
> A folder outline with a subtle overlay of a “project” symbol—like a document, gear, or small stacked squares—so it’s recognizable even at small sizes.
