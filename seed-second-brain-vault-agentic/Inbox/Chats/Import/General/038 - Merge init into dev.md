# Merge init into dev

**Priority:** Medium

## Conversation

### Assistant

You are inside the default editor (**Vim**) for the merge-commit message.

To exit and save the merge commit:

### Save + exit (complete the merge)
1. Press  
   ```
   ESC
   ```
2. Then type:  
   ```
   :wq
   ```
3. Press **Enter**

---

### Exit without saving (cancel merge commit)
1. Press  
   ```
   ESC
   ```
2. Then type:  
   ```
   :q!
   ```
3. Press **Enter**

Use **:wq** if you want the merge to finish.
