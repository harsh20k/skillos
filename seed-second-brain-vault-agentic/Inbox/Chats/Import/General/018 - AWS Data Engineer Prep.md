## **How Your Credentials Affect Salary**

### **Master’s Degree**
- Helps for higher starting salary and eligibility for **mid-level / senior roles** faster.  
- Employers often value advanced degrees especially in data engineering contexts.

### **AWS Certification**
- Employers value AWS cloud certification — **can increase salary by ~10–30%** over non-certified peers in similar roles. citeturn0search29  
- Demonstrates proven cloud skills, particularly in AWS data architecture.
