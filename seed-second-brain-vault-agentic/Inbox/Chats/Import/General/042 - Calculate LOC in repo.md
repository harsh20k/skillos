# How to delete a forked repo on GitHub

1. Go to **https://github.com** and sign in.
2. Open the **forked repository** you want to delete.
3. Click **Settings** (top bar → usually the last tab).
4. Scroll all the way down to the **Danger Zone** section.
5. Find **Delete this repository**.
6. Click **Delete this repository**.
7. GitHub will ask you to type the full repo name (example: `harshpandey/repo-name`).
8. Confirm.

The fork will be **permanently removed** from your GitHub profile.
