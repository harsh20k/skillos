# Add GitHub link LinkedIn

**Priority:** Medium

## Conversation

### Assistant

LinkedIn **always creates a thumbnail** when you add a link in the **Featured** section.  
There is **no option to hide or remove the thumbnail** in that section.

The person in your screenshot did **not** use Featured.

She added her GitHub link in the **About (Summary) section** so it shows as a plain blue hyperlink **without thumbnail**.

---

## **How to add GitHub link WITHOUT a thumbnail (exactly like the screenshot)**

### **Steps**
1. Go to your **LinkedIn Profile**.
2. Scroll to the **About** section.
3. Click **Edit (✏️)**.
4. At the bottom of your About text, add your GitHub link on a separate line:

```
https://github.com/yourusername
```

5. Save.

This displays:
- plain blue link  
- no preview  
- no thumbnail  
- appears directly under your headline (because the About section is very close)

---

## **Optional: also add it to Contact Info (hidden thumbnail)**
1. Click **Contact Info** → **Edit**.
2. Add your GitHub under **Website**.
3. Choose type **Other**.

This adds a link but only visible when users open Contact Info.
