# Change shortcut for Spaces

**Priority:** Medium

## Conversation

### Assistant

On **macOS**, the shortcut to switch Spaces (Desktops) is configurable.

### Change the shortcut for switching Spaces

1. Open **System Settings**
2. Go to **Keyboard**
3. Click **Keyboard Shortcuts…**
4. Select **Mission Control** (left sidebar)
5. Find:
   - **Move left a space**
   - **Move right a space**
6. Double-click the shortcut on the right
7. Press the new key combination you want
8. Close settings (changes apply immediately)

### Notes
- If your shortcut doesn’t register, it’s likely already used elsewhere. macOS won’t allow duplicates.
- Many users prefer:
  - `Option + ← / →`
  - `Cmd + Option + ← / →`
  - `Ctrl + Option + ← / →`

### If it still switches with Ctrl + arrows
Also check:
- **System Settings → Trackpad → More Gestures**
  - Ensure gestures aren’t overriding your intent.
