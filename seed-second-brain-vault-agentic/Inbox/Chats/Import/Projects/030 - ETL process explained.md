# ETL process explained

**Priority:** Medium

## Conversation

### Assistant

ETL means **Extract → Transform → Load**. It is a standard process used to move data from one system into another in a clean, usable form.

**Extract:**  
Pull raw data from source systems  
(databases, APIs, files, logs, sensors, SaaS apps).

**Transform:**  
Clean, validate, join, filter, and reshape the data  
(remove errors, convert types, aggregate, enrich).

**Load:**  
Store the final processed data into a target system  
(data warehouse like Snowflake/BigQuery, or a database).

Used heavily in analytics, reporting, data warehousing, ML pipelines.
