# Java vs JAVA

**Priority:** Medium

## Conversation

### Assistant

The correct way to write **Java** is with an uppercase **J** and the rest in lowercase, like this: **Java**. This is consistent with its official branding and how it's commonly written in technical and professional contexts.

If you're referring to the programming language, always use **Java** unless there's a stylistic or specific reason to use **JAVA**, such as in acronyms or uppercase text formatting.
