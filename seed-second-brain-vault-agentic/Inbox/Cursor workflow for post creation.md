## Conclusion

This workflow is designed to remove all the friction in creating a new post and hence minimising the chance of skipping days and weeks.

If you do not use cursor there are way to set the rules in vscode, claude and gemini.

It will also work if you work with ChatGPT UI but you have to copy/paste the framework file each time.

### Pros
- From thought to final post takes very little time.
- Low effort, which means hard to skip. 
- Still uses your writing style which is good for authenticity.

### Cons
- Content is generated from an initial idea not 100% written. 
- Could be thought of as AI assisted. 
- Removes the craft of writing something.
