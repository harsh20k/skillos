# Idea

I see this app as a companion app to count the naam-japa daily. It should have the following characteristics.
- **Both** on Iphone and Watch.
- It should be **fun** to use and interact with maybe with 3d **animations**.
- It can be inspired with money saving animations, like putting money in gullak.
- **Vault**- how much naam-jap is there in the **vault** **lifetime**.
- Can be inspired with **streaks** app.
- It could have **gaming** like feel that can be used for continuous incrementing like gaming. 
- Since **naam is avinashi**- no possibility to spend the naam-jap.
- **Level syste**m like duolingo.
- **Widgets** and very funny and rediculous reminders and **incessant reminders** to do naam-japa.
- It could be open-sourced.
- For both parents and children.
