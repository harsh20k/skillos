# Podcast Guide Creation Prompt

You are a seasoned podcaster, marketer, and content creator. Create a simple 15-minute podcast guide (not too text-heavy, mostly pointers) for a podcast covering tech, business, networking, life, and content creation.

## Context & Themes

### Tech Themes
- **Kubernetes/Homelab Journey**: 9-day migration from Docker to K8s (pods, deployments, CI/CD, self-healing systems)
- **Agentic AI & RAG**: Retrieval Augmented Generation, embeddings, semantic search
- **Cloud & Data**: AWS, Snowflake data warehousing, serverless architecture
- **Full-Stack Development**: MERN/MEAN stack, Flask, React Native, TDD practices
- **DevOps & Automation**: CI/CD pipelines, database migrations, GitLab workflows

### Business/Networking Themes
- **LinkedIn Strategy**: Weekly content planning, engagement analysis, Nova Scotia employer targeting
- **Career Development**: MACS at Dalhousie, academic roadmap, portfolio building
- **Marine Tech Industry**: Ocean data analytics, Halifax harbour projects, local industry fit
- **Project Management**: User stories, GitLab Issues, documentation practices

### Life Themes
- **Meditation & Rituals**: Structured spiritual practice, habit formation, inner clarity
- **Health Tracking**: Personal health data management
- **Language Learning**: French & Spanish vocabulary building
- **Personal Systems**: Obsidian capture-first workflow, knowledge management

### Content Creation Themes
- **Note-Taking Systems**: Inbox → Projects → Slipbox workflow
- **Learning Documentation**: Project summaries, course notes, reflection practices
- **Group Project Lessons**: TDD adoption, tech stack pivots, automation wins

### New Country & First Semester Themes
- **Relocation Experience**: Arriving in Halifax from Delhi (Sept 2025), climate acclimatization, clean environment appreciation
- **Cultural Adjustments**: Freedom vs responsibility, individual accountability, no "blame game" culture
- **Relationship Dynamics**: Navigating new roles (student + spouse job search), household management shifts
- **First Semester at Dalhousie**: CSCI 5308 industry-simulated projects, weekly client meetings, real-world development lifecycle
- **Learning Outcomes - Academic**: Software engineering is about people, not just code; planning & communication > perfect algorithms
- **Learning Outcomes - Technical**: TDD adoption, Flask backend migration, automation scripts, GitLab Issues as documentation
- **Learning Outcomes - Personal**: Taking initiative in teams, staying professional under pressure, documentation as accountability tool
- **Cost of Living Reality**: PPP comparisons (Canada vs India), TFSA vs RRSP for newcomers, financial planning adjustments

## Output Requirements

Create a **15-minute podcast guide** that:
- Uses bullet points and short phrases (not paragraphs)
- Includes 3-4 main talking points with sub-pointers
- Has natural transitions between segments
- Includes 1-2 engaging questions/hooks to open
- Suggests 1-2 actionable takeaways for listeners
- Keeps tone conversational and practical
- Mixes personal stories with broader insights

## Format Structure

```
Opening Hook (30 sec)
- [Engaging question or story]

Segment 1: [Theme] (4-5 min)
- Point 1
- Point 2
- Point 3

Segment 2: [Theme] (4-5 min)
- Point 1
- Point 2
- Point 3

Segment 3: [Theme] (4-5 min)
- Point 1
- Point 2
- Point 3

Closing & CTA (1 min)
- Key takeaway
- Call to action
```

**Note**: Draw from the themes above but don't feel limited—use your expertise to create engaging, valuable content that resonates with tech professionals, students, and content creators.
