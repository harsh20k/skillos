While deploying my first cloud project I got deep into infrastructure as code. That led me to a Python library that generates architecture diagrams from code. I added it as a skill in Cursor now I describe what I want in plain language and get a diagram in seconds.

There’s still friction. Tweaks and iterations take time. But the jump from “describe it” to “see it” is real.

It sharpened my view on AI in dev tools: it’s going to be a basic tool in every developer’s hand. Using it instead of learning fundamentals is a mistake. Once you have the basics, it’s powerful for brainstorming, exploring new ideas, and trying solutions you wouldn’t have drawn by hand. It’s never been easier or more fun, to go from a rough idea to a clear picture.

- Hook: First cloud deploy → IaC → Python diagrams-from-code
- Cursor skill: natural language → diagram in seconds
- Honest limit: iteration and tweaks still take time
