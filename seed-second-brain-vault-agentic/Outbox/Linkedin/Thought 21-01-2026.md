Microsoft's [$19B CAD AI investment](https://www.msn.com/en-ca/technology/cybersecurity/microsoft-s-ai-deal-promises-canada-digital-sovereignty-but-is-that-a-pledge-they-can-keep/ar-AA1UrZvE) promises Canadian data sovereignty. But can we trust promises alone?

I'm learning Infrastructure as Code because real sovereignty isn't just where data sits—it's knowing how to build and control the stack yourself.

Software Defined Data Centers aren't just technical; they're practical independence. When you can define, version, and recreate entire environments from code, you're not locked into any vendor's vision.

The deal raises a question: if Microsoft can keep data in Canada yet remain subject to foreign laws like the CLOUD Act, what does sovereignty actually mean?

Maybe true sovereignty starts with understanding the infrastructure—not just consuming it.

Who else is thinking about the technical skills needed for genuine digital independence?
